# Panoramic Plumbing Website V2

A modern, sleek, and professional website for Panoramic Plumbing - a third-generation plumbing company serving Manchester, NH and surrounding areas.

## Features

- **Modern Bank-Like Design**: Clean, professional, and trustworthy appearance with a blue color scheme
- **Responsive Layout**: Fully responsive design that works on all devices (desktop, tablet, mobile)
- **Unified Layout System**: Consistent header and footer across all pages
- **Smooth Animations**: Subtle, professional animations that enhance user experience without being distracting
- **SEO Optimized**: Proper meta tags, semantic HTML, and optimized content
- **Fast Loading**: Optimized CSS and JavaScript for quick page loads

## Pages

1. **Home (index.html)**: Main landing page with services, about section, reviews, and contact information
2. **Terms of Service (terms-of-service.html)**: Complete terms and conditions for services
3. **Privacy Policy (privacy-policy.html)**: Comprehensive privacy policy

## Technology Stack

- **HTML5**: Semantic markup for better SEO and accessibility
- **CSS3**: Modern styling with CSS Grid, Flexbox, and animations
- **Vanilla JavaScript**: No dependencies, fast and lightweight

## Features Implemented

### Header
- Sticky navigation that stays at the top when scrolling
- Contact information bar (phone and email)
- Mobile-responsive hamburger menu
- Smooth animations on load

### Hero Section
- Gradient background with animated grid pattern
- Clear call-to-action buttons
- Engaging headline and description

### Services Section
- Four main service categories with icons
- Hover effects on cards
- Grid layout that adapts to screen size

### About Section
- Company information and values
- Three highlighted features (Locally Owned, Third Generation, Quality Guaranteed)
- Customer testimonials

### Reviews Section
- Real customer reviews from Google
- Star ratings
- Link to leave a review

### Footer
- Company information
- Service links
- Contact information
- Legal links (Terms & Privacy)

### Animations
- Smooth fade-in and slide animations
- Hover effects on cards and buttons
- Mobile menu transitions
- No scroll-triggered pop-in effects (as requested)

## How to Use

1. Open `index.html` in a web browser to view the homepage
2. Navigate through the site using the menu
3. All pages share the same header and footer for consistency
4. The site is fully functional and ready to deploy
